# Rock Paper Scissors Pentagon Game

A modern, fully responsive Rock-Paper-Scissors game built with HTML, CSS, and JavaScript. This project features a pentagon layout for the game choices, animated results, and a rules modal.

## Features

- **Pentagon Choice Layout:** Unique positioning of the Rock, Paper, Scissors, Lizard, and Spock buttons.  
- **Score Counter:** Keeps track of the player's score.  
- **Winner Highlight Animation:** The winning choice is highlighted with a smooth animation.  
- **Rules Modal:** Explains how the game works.  
- **Fully Responsive:** Works on desktops, tablets, and mobile devices.  
- **Custom Design:** Gradient backgrounds, shadows, and modern typography using Google Fonts.  

---

## Rules & Bonus

**Scoring:**  
- If the player wins, they gain 1 point.  
- If the computer wins, the player loses 1 point.  

**Bonus Rules:**  
- Scissors beats Paper  
- Paper beats Rock  
- Rock beats Lizard  
- Lizard beats Spock  
- Spock beats Scissors  
- Scissors beats Lizard  
- Paper beats Spock  
- Rock beats Scissors  
- Lizard beats Paper  
- Spock beats Rock
